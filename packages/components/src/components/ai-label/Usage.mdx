# AI Label
The AI Label highlights AI features or AI generated content. For further information on Usage and Design Patterns, see the links below.<br />
There are two different variants of the AI Label: use **either** the AI Icon **or** the Text Label depending on where clarity is needed most. The divider is also removable, see Figma's design pattern page for guidance.
