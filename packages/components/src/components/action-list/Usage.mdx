# Action List

The Action List component is a container that displays a collection of actionable items. Each item can contain leading content (icons, images, or interactive elements), title text, description text, and trailing content (typically a chevron or interactive element).

## Interactive Elements

When placing interactive elements in the leading or trailing slots, the following Infineon components are supported and will prevent the main item click event from firing when interacted with:

- **IFX-Button** - Interactive buttons
- **IFX-Checkbox** - Checkbox controls
- **IFX-Switch** - Toggle switches

All other elements will trigger the main item click event when activated.
