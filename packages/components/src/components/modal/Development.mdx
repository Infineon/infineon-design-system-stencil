## Notes
The Modal component is a modal window that appears on top of the app content and allows users to interact with it. 

Example in VanillaJs:

```js
<ifx-modal caption="Modal Title" variant="alert-brand" close-on-overlay-click="true">
  <div slot="content"><span>Modal content</span></div>
  <div slot="buttons">
    <ifx-button variant="secondary" onclick="console.log('Button 1 clicked')">Button 1</ifx-button>
    <ifx-button onclick="modal.close()">Button 2</ifx-button>
  </div>
</ifx-modal>
```

For more usage examples within other Javascript frameworks, please check the example applications that are linked in the ```README.md``` of our [Github repository](https://infineon.github.io/infineon-design-system-stencil).
